# Server for Redux Demo

## Setup

```
$ npm install
$ npm start
```

That should get you a dev server at `http://localhost:8080`

## Available Methods

### `GET /`

Returns `{ count: x }`

### `POST /increment`

Increments count by 1 and returns `{ count: x }`

### `POST /decrement`

Decrements count by 1 and returns `{ count: x }`
